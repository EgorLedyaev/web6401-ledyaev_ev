# Лабораторная работа №2 — HTML и CSS

Тема: туристический сайт «Мини‑гайд по городу Москва».

## Структура проекта

```
lab2/
├─ index.html        # Главная страница
├─ sights.html       # Достопримечательности
├─ food.html         # Еда
├─ plan.html         # План визита (таблица с выравниванием)
├─ styles.css        # Общий подключаемый CSS
└─ assets/           # Фотографии (JPEG)
   ├─ blini2.jpg
   ├─ bolshoi.jpg
   ├─ borscht.jpg
   ├─ kremlin.jpg
   ├─ pelmeni.jpg
   ├─ pirozhki.jpg
   ├─ red_square.jpg
   ├─ st_basils.jpg
   └─ vdnh.jpg
```

## Выполненные требования

- Количество страниц: 4 (от 3 до 5).
- HTML:
  - Использованы блочные элементы (`header`, `section`, `div`, `main`, `footer`) и строчные (`a`, `span`, `em`, `strong`, `img`).
  - Ссылки: навигация между страницами + внешние ссылки на официальные ресурсы.
  - Картинки: на страницах «Главная», «Достопримечательности», «Еда» присутствуют изображения (фото с Wikimedia Commons, JPEG).
  - Таблица: на странице `plan.html` с выравниваниями.
- CSS:
  - Отдельный подключаемый файл `styles.css`.
  - Минимум 3 видов селекторов: типовой (`body`, `h1`), класс (`.card`, `.btn`), id (`#hero`), потомка (`.logo span`, `.card > h3`), псевдокласс (`a:hover`, `.site-nav a.active`), атрибутный (`a[target="_blank"]::after`).
  - Цвета: заданы через переменные и используются в тексте, фонах, кнопках.
  - Таблица: классы `.align-left`, `.align-center`, `.align-right` для выравнивания по левому/правому краю и по центру.
- Запреты: не применялись JS, Bootstrap и другие фреймворки.

## Оптимизация

- Lazy‑loading изображений: всем `img` добавлены `loading="lazy"` и `decoding="async"` для ускорения загрузки страниц.
- Выровнены фото в карточках: через CSS `aspect-ratio` и `object-fit: cover` для единообразной сетки и снижения визуальных скачков при загрузке.
- Рекомендации (по желанию):
  - Сжать JPG до ширины ~1200px и качества 75–85% для уменьшения веса.
  - Добавить `srcset/sizes` для адаптивных изображений.

## Как запустить локально

1. Откройте файл `lab2/index.html` напрямую в браузере, или
2. Поднимите локальный сервер:
   - Python: `python -m http.server` в папке `lab2`, затем перейти на `http://localhost:8000/`.
