## Лабораторная работа №4

Цель: изучить современные возможности CSS (Flexbox, Grid, переменные, анимации) и основы ООП на JavaScript.

План выполнения:
- Создана ветка `lab4` и папка `lab4` на базе исходников `lab2`.
- Модернизированы страницы:
  - `index.html` — сетка карточек на CSS Grid, анимация «пульс» для кнопки.
  - `food.html` — карточки блюд сверстаны на Flexbox.
  - `sights.html`, `plan.html` — обновлённые стили, переменные цветов.
  - `form.html` — страница с формой (логин, пароль, чекбокс) и JS‑классом.
- В `styles.css` добавлены CSS‑переменные, Flexbox для шапки, анимации.
- BEM применён для формы: блок `form`, элементы `form__input`, `form__label`, `form__submit`.

После проверки — создать Pull Request из `lab4` в `main`.
